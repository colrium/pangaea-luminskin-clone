Pangaea Frontend Takeaway Challenge built on React and Next.js

## Getting started

### Development

`yarn install` or `npm install`;
`yarn dev`;
open `http://localhost:3000`

### Testing

`yarn test`

### Lint

`yarn lint`

### Type check

`yarn type-check`;

### Prettier

`yarn format`
